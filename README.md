# Система анализа продаж (Sales Analysis System)

Проект представляет собой систему для анализа данных о продажах, написанную на Python. Система позволяет загружать, очищать, анализировать и визуализировать данные о продажах, а также экспортировать результаты в удобном формате.

Основные возможности:
    Загрузка данных из CSV-файлов с поддержкой различных кодировок
    Очистка и предобработка данных (обработка пропусков, преобразование типов)
    Комплексный анализ продаж по различным измерениям:
        По категориям товаров
        По регионам
        По продавцам
        По временным периодам
    Визуализация результатов в виде графиков и диаграмм
    Экспорт результатов в Excel или CSV форматах
    Генерация тестовых данных для демонстрации работы системы

Требования
Системные требования
    Python 3.8 или выше
    4 ГБ оперативной памяти (рекомендуется)
Установка зависимостей
```bash
pip install pandas numpy matplotlib openpyxl
```
Или используйте файл requirements.txt:
```bash
pip install -r requirements.txt
```
Опциональные зависимости
    openpyxl - для экспорта в Excel формат (устанавливается автоматически)
Быстрый старт
1. Клонирование и запуск
```bash
# Клонируйте репозиторий (если есть)
git clone <repository-url>
cd sales-analysis-system
# Запустите систему
python main.py
```
2. Использование существующих данных
    Поместите ваш CSV-файл с данными о продажах в папку проекта
    Запустите python main.py
    Выберите опцию "1. Использовать существующий файл"
    Выберите ваш файл из списка или введите путь к нему
3. Генерация тестовых данных
```bash
python main.py
```
# Выберите опцию "2. Создать тестовые данные и проанализировать"
Формат входных данных:
Система ожидает CSV-файл со следующими обязательными колонками:

Колонка	Тип данных	Описание

Product_ID	Строка	Уникальный идентификатор товара

Sale_Date	Дата	Дата продажи (YYYY-MM-DD)

Sales_Rep	Строка	Имя продавца

Region	Строка	Регион продажи

Sales_Amount	Число	Сумма продажи

Quantity_Sold	Число	Количество проданных единиц

Product_Category	Строка	Категория товара

Опциональные колонки:

    Unit_Cost - Себестоимость единицы товара

    Unit_Price - Цена продажи единицы товара
    
    Customer_Type - Тип клиента (New/Returning)
    
    Discount - Скидка в процентах
    
    Payment_Method - Способ оплаты
    
    Sales_Channel - Канал продаж

Процесс анализа:

Система выполняет анализ в 5 этапов:

    Загрузка данных - чтение CSV-файла

    Очистка данных - обработка пропусков, преобразование типов

    Анализ данных - расчёт статистик по категориям, регионам и продавцам

    Визуализация - создание графиков и диаграмм

    Экспорт результатов - сохранение в Excel/CSV форматах

        summary_YYYYMMDD_HHMMSS.txt - текстовый сводный отчет
